全局说明

接口请求地址

测试Host：http://db.admin.test.mofang.com/api/card/
正式Host：http://db.admin.mofang.com/api/card/

所有请求返回数据类型均为JSON

{
    code : 0 (0：成功，其他失败),
    data : 响应信息
}

API列表

1 获取选择字段的已使用选项

  • 请求地址:/getitems
  • 请求方式: get
  • 请求参数:
参数 是否为必填项 含义 示例
setid true 游戏表id 1
select flase 返回字段逗号分隔 name_gq,listorder
filter false 过滤条件 djfl|珍品
regex false 正则匹配 djname|碎片
order false 排序，默认id正序 xyd|1
page false 当前页码，默认1 1
size false 每页数量,默认不分页 20
  • 响应结果:
{
	"code":0,
	"data":[
		{"id":497,"data":{"name_gq":"序章"},"listorder":0},
		{"id":498,"data":{"name_gq":"日本区域1"},"listorder":0},{"id":499,"data":{"name_gq":"日本区域2"},"listorder":0},
		...
	],
	"pages":{"itemCount":22,"pageSize":0,"currPage":1}
}

2 获取选择框的候选项

  • 请求地址:/getoptionlist
  • 请求方式: get
  • 请求参数:
参数 是否为必填项 含义 示例
setid true 游戏表id 1
field true 字段英文名 xj
  • 响应结果:
{
    "code":0,
    "data":["1星","2星","3星","4星","5星","6星"]
}
